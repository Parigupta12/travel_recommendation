# Travel Recommendation Web App 🌍

This is a simple travel recommendation website created using HTML, CSS, and JavaScript.

## Features:
- Home, About, and Contact pages
- Navigation bar
- Search functionality
- Beach, Temple, and Country recommendations
- Images display with results
